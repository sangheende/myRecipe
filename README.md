## ✨ 주요 기능

- 레시피 등록 (재료, 조리 순서, 소요 시간)
- 카테고리별 분류 — 밥 🍚 / 면 🍜 / 국·찌개 🍲 / 반찬 🥢 / 구이·튀김 🍳 / 빵·간식 🍞 / 음료 🥤
- 레시피 이름 및 재료로 검색
- 관리자 모드에서 레시피 추가·수정·삭제 (`?admin=true`)
